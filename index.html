HTML & CSS Cheat Sheet

HTML Boilerplate Tags

Tag Description Use
< ! DOCTYPE html > Declares version of html for browser to interpret. Used once at beginning of document.
< html > Establishes language being used is html. Opens & closes document.
< head > Non-visible data that contains information like titles and links. Used only in the opening section of a document.
< title > Name that will appear in the window or tab. Used only within the < head > section of the document.
< body > Establishes the main body of code. Content that is visible on the webpage. Opens & closes document. Used within < html > tags.

Semantic HTML Tags

Tag Description Use
< nav > Navigation section Contains links to navigate a document.
< header > Specifies a header for a document or section. Can be used throughout a document any time there is a header
< main > Specifies the main content of a document Used at open and close of document.
< section > Defines a section in a document. Used for a thematic grouping of content, typically with a heading. (EX: Chapters, introduction, news items, contact info...
< article > Specifies independent, self-contained content An article should make sense on its own, and it should be possible to distribute it independently from the rest of the web site. (EX: blog/forum posts, comments, newspaper articles...
< p > Defines a paragraph. Used to separate out chunks of information.
< aside > Defines some content aside from the content it is placed in (like a sidebar) Aside content should be indirectly related to the surrounding content.
< footer > Defines a footer for a document or section Typically contains author info, copyright info, contact info, sitemap, links back to top, related documents...

CSS Box Model

Property Description How to Use
Content The content of the box, where text and images appear
Padding The padding is the spacing between the content and the border (edge of the content). padding: xpx; (reads clockwise starting at top)
Border A border that goes around the padding and content border: xpx solid/dotted/etc color;
Margin The margin is the space around the outside of the border. The larger the margin, the more space between our element and the elements around it. margin: xpx; (reads clockwise starting at top)
